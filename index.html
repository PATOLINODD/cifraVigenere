Cifra de VIGENERE

IGENEREV

A cifra de Vigenère é como um código secreto para esconder mensagens

pense assim:

Mensagem: GATO


Chave: LUA


Mensagem: G A T O

Chave:    L U A L

A = 0, B = 1, C = 2, D = 3, E = 4, F = 5, G = 6, H = 7, I = 8, J = 9, K = 10, L = 11, M = 12,
N = 13, O = 14, P = 15, Q = 16, R = 17, S = 18,
T = 19, U = 20, V = 21, W = 22, X = 23, Y = 24,
Z = 25

G = 6, A = 0, T = 19, O = 14

L = 11, U = 20, A = 0

Começa com G
G (06) + L (11) → chega no R
A (00) + U (20) → chega no U
T (19) + A (00) → continua T
O (14) + L (11) → chega no Z

RUTZ

Só quem souber a palavra secreta (LUA) consegue voltar para GATO.
